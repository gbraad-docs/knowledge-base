# Audio

Notes on audio programming tools, environments, and projects.

- [Pure Data](puredata.md) - visual dataflow programming for audio and multimedia
- [Csound](csound.md) - orchestra/score language for sound synthesis and processing
- [Faust](faust.md) - functional DSP language that compiles to standalone C; targets Logue SDK, WebAssembly, and more
- [Logue](logue.md) - Logue SDK effect units for the KORG NTS-1 / minilogue xd
- [Schwung/Move effects](schwung-move-effects.md) - DSP effects for the Ableton Move (using Schwung)
